1. Integrate angular-wakanda in your project.

Thanks to Bower, install angular-wakanda in a command line or via your own bowser.json file. 

bower install angular-wakanda

If you want to install angular-wakanda manually, you just need to include angular-wankanda-min.js, which itself requires angular.min.js. 

<script src="/bower_components/angular/angular.min.js"></script>
<script src="/bower_components/angular-wakanda/angular-wakanda.min.js"></script>

2. Initialize the $wakanda service.

The init() method loads all the dataclasses in the Wakanda Model (by default) or only the dataclasses you specify and returns a JavaScript promise.

This Wakanda project's model contains three dataclasses: Country, Company, and Employee.

A Wakanda dataclass model can be designed either graphically or programmatically in pure JavaScript.

3. Retrieve your data with one line of code.

angular-wakanda lets you use your entire Wakanda model directly from Angular-based Web Application interfaces.

Once the $wakanda service is initialized, you can integrate your previously defined business JS objects into your angular scopes simply by calling an angular-wakanda method, like $find():

$scope.countries = ds.Country.$find();

4. Get fast infinite scrolling.

With angular-wakanda, you don't need to manually manage paging or cache optimization.

angular-wakanda takes advatange of Wakanda's native entityset cache to fetch a large collection of entities in pages.

You can either use the $more() method for infinite scrolling or the classic $prevPage() and $nextPage() methods.

<ul id="list" when-scrolled="employees.$more()">
	<li ng-repeat="employee in employees">{{employee.fullName}}</li>
</ul>

5. Retrieve related data automatically.

No joins, no foreign keys, and no map-reduce.

Get access to related data directly through your entity object properties.

All you have to do is call the $fetch() method to load the related attribute's data locally.

// get companies related to current country
country.companies.$fetch();

Thanks to Wakanda's alias attributes, this example shows the manager's fullname without even fetching the related manager.

6. Modify your data.

Each Wakanda dataclass provides a $create() method for you to create a new entity.

Each Wakanda entity comes with $save() and $remove() methods.

<strong>Salary</strong>: <input ng-model="employee.salary" /><br>
<input type="button" value="save" ng-click="employee.$save()">
